HTML

HTML é uma linguagem de marcaçãoe não de programação ja que ele não permite criar funcionalidade dinamicas, que permite a construção de paginas WEB, e tambem utilizada para aplicativos mobile, sendo possivel cria varias paginas diferentes, sem falar que atualmente HTML se tornou o padrão da internet então você com certeza ja viu um site em HTML. O HTML é uma linguagem amigável para iniciantes, que possui bastante suporte e é principalmente usada para páginas estáticas. O HTML funciona melhor quando usado em conjunto com o CSS para aplicar o estilo e com o JavaScript para implementar funcionalidades e sendo seu unico limite para se cria algo, sua imaginação e o quanto você vai se focar e dedicar para aprender HTML.

CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas, O CSS é aquilo que vai dar personalidade a sua pagina, não sendo uma linguagem de programação. É um mecanismo que permite o programador por meio de milhares de tags para cada coisa, deixar a pagina no estilo que mais o agradar, A relação entre HTML e CSS é bem forte. Como o HTML é uma linguagem de marcação e o CSS é focado no estilo, eles andam juntos. lembrando que mais uma vez o unico limite aqui é a imaginação e o tempo que você vai se comprometer a apreender

TAGS

As tags são os elementos utilizados para marcações, sendo cada uma digitada de for especifica, contendo um nome para cada uma.

Como por exemplo o "body" que representa o corpo do seu codigo, e consequentemente o corpo do seu site; Tudo que for digitado aqui, se estiver de acordo com as diferente tags aparecera no seu site. Outras tags importantes são as tags de texto que vão de "h1" até "h5"(se não me engano), e essas tags são feitas para conterem muitas vezes texto nelas sendo que quanto maior o numero na frente do "h" menor sua letra vai ficar, sendo "h1" a maior de tidas, mas não são só elas que servem para texto; Temos tambem a tag "p" na qual você esta lendo esse texto, entre outras para puxar imagens, videos, abrir uma seção, abrir uma divisoria, criar um botão, etc.
